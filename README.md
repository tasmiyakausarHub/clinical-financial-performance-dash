Project Overview

This project presents an interactive Hospital Operations Dashboard built entirely using Tableau to analyze both clinical performance and financial metrics of a healthcare organization.

The dashboard enables healthcare administrators and decision-makers to monitor hospital operations, improve patient care, and optimize financial performance through data-driven insights.

🎯 Objectives
Analyze patient flow and hospital utilization
Track clinical performance indicators
Monitor financial metrics such as revenue and expenses
Identify operational inefficiencies
Deliver interactive visual insights using Tableau
📊 Key Features
📌 Patient Admissions & Discharges Overview
📌 Bed Occupancy Rate Analysis
📌 Department-wise Performance Dashboard
📌 Revenue vs Expenses Visualization
📌 Patient Demographics Insights
📌 Treatment & Diagnosis Trends
📌 KPI Indicators for quick decision-making
🛠️ Tools & Technologies Used
Tableau Desktop – Data visualization & dashboard creation
Tableau Public (optional) – Dashboard publishing
Excel / CSV – Data source and preprocessing
📂 Dataset Description

The dataset includes:

Patient details (age, gender, admission type)
Admission & discharge records
Department information
Treatment & diagnosis data
Billing and financial records

Note: This project uses sample/simulated healthcare data for analysis.

📈 Dashboard Insights
Identified departments with high patient load
Analyzed peak admission periods
Evaluated revenue contribution by department
Detected inefficiencies in resource utilization
